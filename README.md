EigenFace
=========

EigenFace implementation in Python for Yale FaceDatabase

### Requirements
- Numpy
- Matplotlib
- PIL

You could see what an EigenFace is and how the code works in [here][0]. The [ipython notebook][1] in the directory shows
the flow of the program.
